# Go / React starter app

## Description
This is a boiler plate hello world app for building a web application using
Golang on the server side and React on the client side.

## Requirements:
* golang 1.8
* npm
* jspm

## Usage
* `go get github.com/titolins/go_react_starter`
* `cd client`
* `npm install`
* `jspm install`
* `cd ../server`
* `go run main.go`
